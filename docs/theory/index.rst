Overview
--------

The python_magnetcooling package implements various theoretical models for thermal-hydraulic analysis
of water-cooled systems, particularly for high-field magnets.

Key Topics
----------

* **Heat Transfer**: Convective heat transfer correlations, Nusselt number relationships
* **Fluid Dynamics**: Friction factors, pressure drop calculations, flow regimes
* **Thermodynamics**: Water properties (IAPWS), energy balance equations
