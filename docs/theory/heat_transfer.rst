Correlations
------------

Gnielinski Correlation
~~~~~~~~~~~~~~~~~~~~~~

For turbulent flow in pipes:

.. math::

   Nu = \frac{(f/8)(Re-1000)Pr}{1 + 12.7(f/8)^{0.5}(Pr^{2/3}-1)}

Valid for:
* :math:`3000 < Re < 5 \times 10^6`
* :math:`0.5 < Pr < 2000`

Dittus-Boelter Correlation
~~~~~~~~~~~~~~~~~~~~~~~~~~~

Simplified correlation for turbulent flow:

.. math::

   Nu = 0.023 Re^{0.8} Pr^{n}

where :math:`n = 0.4` for heating and :math:`n = 0.3` for cooling.

Valid for:
* :math:`Re > 10000`
* :math:`0.7 < Pr < 160`
